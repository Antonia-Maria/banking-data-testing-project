# Banking Data Testing Project

## Overview
This is a **beginner-friendly learning project** focused on **data testing using SQL and Python**.
The project uses a simplified **banking domain** (customers, accounts, transactions) to practice data modeling, SQL design, and automated testing.


The goal is to learn **how to test data correctly**, not to build a full application.

---

- SQL
- Python
- pytest
